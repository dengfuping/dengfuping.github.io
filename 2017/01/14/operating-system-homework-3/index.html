作业题目

有一个仓库，可以存放 A 和 B 两种产品，仓库的存储空间足够大，但要求：
（1）一次只能存入一种产品（A 或 B）；
（2）-N < (A 产品数量 - B 产品数量) < M。
其中， N 和 M 是正整数。试在POSIX系统平台上实现产品A和产品B的入库过程。
